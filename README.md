# Armstrong AI editing — POC demo

Technical proof of concept for an AI-assisted rough-cut tool, built for [Mat Armstrong](https://www.youtube.com/@MatArmstrong)'s automotive YouTube channel.

Live demo: https://re-thrive-matt.github.io/armstrong-ai-edit-poc/

This is a *proof of concept*, not a finished product — it demonstrates that the underlying AI (transcription, audio-event classification, LLM chapter writing) works on workshop-vlog content. Turning it into a daily-use tool is a separate build.
